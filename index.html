Resumen profesional

Data Automation / BI Engineer con experiencia en AYESA para proyectos de Endesa, centrado en automatizar flujos de datos y reporting en entornos corporativos.

Trabajo con Python, SQL, SOQL y ETL para integrar fuentes y generar datasets robustos para Power BI, aplicando controles de data quality en procesos recurrentes.

Experiencia en explotación de Salesforce, Redshift y otras plataformas para transformar datos en entregables útiles para negocio y operación.

Busco equipo técnico con revisión de código y estándares.

Propuesta de valor

  • Automatización y estandarización de reporting y procesos ETL para eliminar tareas manuales.
  • Integración de fuentes corporativas con Salesforce/SOQL, Redshift y FTP para consolidar datos.
  • Data quality mediante validación y control de consistencia en cada entrega de datos.
  • Power BI con datasets estables, modelado y dashboards orientados a seguimiento operativo.

Experiencia profesional

AYESA — Proyectos Endesa (Sep 2019 – Actualidad)

Endesa Distribución (2019–2024)

  • Diseñé extracciones recurrentes con SQL para reporting mensual, garantizando entregas periódicas para áreas operativas.
  • Desarrollé consultas ad-hoc con SQL para responder peticiones puntuales sin romper el flujo de reporting estándar.
  • Implementé validaciones de calidad de datos sobre nulos, duplicados y coherencia de campos para mejorar la fiabilidad de los resultados.
  • Revisé datos de entrada y corregí incidencias de calidad para evitar errores en informes y retrabajos posteriores.
  • Preparé informes y ficheros de entrega en Excel, XML y formatos requeridos para asegurar compatibilidad con distintos consumidores.
  • Estandaricé ejecuciones periódicas de consultas y generación de salidas para mantener continuidad operativa.
  • Documenté reglas de extracción y validación para facilitar traspaso de conocimiento y mantenimiento técnico.

Aportación / Resultado

  • Reducción de tareas manuales y errores en procesos de reporting recurrente.
  • Mayor trazabilidad y reproducibilidad en extracciones y entregables.
  • Mejor mantenibilidad mediante estandarización de consultas y reglas de validación.

Endesa X (Marketing) (2024–Actualidad)

  • Automatizé procesos ETL con Python para extracción, transformación y carga de datos en flujos de trabajo recurrentes.
  • Automatizé descargas automáticas FTP con Python para asegurar disponibilidad de archivos en ventanas de ejecución periódicas.
  • Generé datasets para Power BI desde múltiples fuentes, preparando estructuras estables para modelado y análisis.
  • Ejecuté consultas SQL y SOQL sobre ATENEA, XCUSTOMER y Salesforce para consolidar información de negocio y marketing.
  • Integré extracciones con Power Automate y Python cuando aplicó para conectar tareas de captura, transformación y entrega.
  • Modelé y transformé datos en Power BI, construyendo dashboards y medidas con DAX intermedio para seguimiento operativo.
  • Refactoricé código y construí funciones reutilizables para mejorar legibilidad, mantenibilidad y continuidad del desarrollo.
  • Elaboré manuales de usuario y empaqueté soluciones como ejecutables para distribución a usuarios no técnicos.
  • Coordiné seguimiento semanal con cliente y analicé nuevas peticiones para convertir necesidades funcionales en tareas técnicas.

Aportación / Resultado

  • Reducción de tareas manuales y errores en procesos de extracción y reporting.
  • Mayor trazabilidad y reproducibilidad en pipelines y preparación de datasets.
  • Mejor mantenibilidad mediante refactorización y reutilización de componentes.
